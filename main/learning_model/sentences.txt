# 가령: 260422: 수정 내용 - 문장 학습용 시드 데이터 추가. 좌측: 수어 어순 단어(콤마 구분) / 우측: 한국어 문장
# 형식: <단어1>, <단어2>, ... = <한국어 문장>
저, 꿈꾸다, 프로그래머 = 저는 프로그래머를 꿈꿉니다
웹사이트, 경험, 만들다 = 웹 사이트 제작 경험이 있습니다
결과, 언제, 알다, 묻다 = 결과는 언제 알 수 있을까요?
안녕하세요, 나, 매일, 성장, 노력, 개발자 = 안녕하세요. 저는 매일 성장하는 꾸준함이 강점인 개발자입니다
상대, 의견, 듣다, 서로, 해결, 노력 = 상대방의 의견을 경청하고 대화를 통해 최선의 합의점을 찾아내려 노력합니다
같이, 사람, 신뢰, 인정, 원하다 = 팀원들에게 인정받고 신뢰할 수 있는 동료가 되고 싶습니다
오늘, 나, 능력, 보여주다, 감사, 다시, 만나다 = 오늘 제 역량을 보여드릴 기회를 주셔서 정말 감사합니다
